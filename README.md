# Fabian Henry - Portfolio

Personal portfolio and engineering showcase built with Next.js and TailwindCSS.

## Stack
- Framework: Next.js 16
- Styling: TailwindCSS

## Structure
- `/` - Main landing page with about, featured work, and experience sections.
- `/projects` - Project catalog with search and technical details in modals.
- `src/data/projects.js` - Centralized metadata for all projects.
